# khinsider-dl
This mass downloads khinsider albums. You just need the album page URL and it just downloads them all.
Only does MP3 (for now)

## Usage Example
'''
khinsider-dl.exe https://downloads.khinsider.com/game-soundtracks/album/minecraft-classic-soundtrack-gamerip-2013 -o ./output/
'''
Without the -o argument by default it just downloads to what directory you run it in.


made on python 3.12.3
